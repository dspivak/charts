It was shown by Ahman and Uustalu \cite{AhmanUustalu} that comonoids $(c,\epsilon,\delta)$
\[
	c\To{\epsilon}\yon
	\qqand
	c\To{\delta}c\tri c
\]

Given any such bicomodule $p$, we have a map
\[
p\To{\lambda}c\tri p\To{\cong}c\tri p\tri \yon\To{c\tri p\tri !}c\tri p\tri\ord{1}
\]
